# Weather Station System Setup

## Overview

This guide explains how to set up a Weather Station using industrial RSpi and other devices to access weather information from ANL such as atmospheric temperatures, pressure, humdity, wind speed, wind direction, and precipitation.

> **Note:** 

---

## Hardware Required

- IND 90W BT PoE Splitter
- RPI CM4101008 INDUSTRIAL PC
- GbE 90W poE++ Injector Std/Legac
- 2.5MM DC PLUG TO FREE END 1000MM
- Access to a SAGE Thor Node
